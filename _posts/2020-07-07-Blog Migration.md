Blog Migration from Hexo to Jekyll

<!-- slide -->

Due to long time no writing blogs and the update with newest Node.js breaks the compilability of my old blog site built with Hexo. So that I want to shift it to a new frame, Jekyll.

<!-- slide vertical=true -->

Once I ditched Hexo and went into Jekyll, the first thing I did was looking for an appropriate theme which can render Math Equations and be able to hold and highlight some code blocks, since my major is Astrophysical Physics, the requisite for Math rendering and code view is indispensable.

<!-- slide -->

But it was difficult to find such a theme, until I discovered this one in a quick glance at the Jekyll theme gallery which perfectly matches my needs. Before I found this theme, I tried to embed MathJax and KeTax in some themes and it just not working.

<!-- slide vertical=true -->

Finally, I met this one, weeb but geek theme, perfectly solves Math rendering and has a beautiful code view. I really appreciate the developer of this theme.

<!-- slide -->

Thanks !
